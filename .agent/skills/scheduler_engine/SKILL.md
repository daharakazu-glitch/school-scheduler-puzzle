# Scheduler Engine Skill

## 責務
- 非常勤講師の勤務可能曜日・時間帯の制約違反検知
- クラス間（A組・B組など）の同一科目・同一時間帯の重複検知
- 変更に伴うカスケードエラーの計算と報告

## 実装ガイドライン
- 制約は純粋な関数として実装し、入力（時間割全体と移動対象のコマ）から無効な移動を真偽値やエラーメッセージの配列として返すこと。
- ReactのContext API等でグローバルなエラー状態を保持し、UIで即座に警告（赤枠など）を表示する。
