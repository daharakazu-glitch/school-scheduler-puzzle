# 📜 MISSION CHARTER (School Scheduler)
**Status:** ACTIVE
**Scope:** ALL PROCESSES

第1条：言語規定
* 日本語を標準とし、コードのコメントも丁寧な日本語で記述する。

第2条：制約充足アルゴリズムの厳守
* 非常勤講師の「勤務日限定」および「習熟度別並行授業の同期」を最優先の制約とする。
* 1つのコマを動かす際、影響を受ける全てのコマを再計算または警告表示するロジックを組む。

第3条：Gitコミット運用
* 作業の節目（コンポーネント作成、ロジック修正等）ごとに、詳細な日本語のコミットメッセージと共に `git commit` を実行する。
* 必要に応じてリモートリポジトリへ `git push` を提案する。

第4条：UI/UX 指針
* ドラッグ＆ドロップによる直感的な操作と、制約違反時の「視覚的な警告」を重視する。
